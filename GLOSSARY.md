# VAS Translation Glossary

The canonical terminology reference for translating VAS / 深握計畫 site content across **繁體中文 (root)**, **English**, **Japanese**, and **Simplified Chinese (Mainland)**.

> **Last updated:** 2026-05-17 · End of major translation sprint (echo / faq / self / context / harness 三本柱).
> **Maintainer:** Nova.
> **Why this exists:** VAS is a literary site, not a typical product page. Many concepts (Jungian, Daoist, Human Design) need consistent handling. This glossary locks the decisions so future translators (human or AI) don't drift.

## 6 · JA language strategy

**Kanji-rich register**: Prefer kanji over katakana where natural. Preserve 協作 / 截圖 (when in body-content, not tech-button) / 人機 etc.

**Modern shinjitai** (use these, NOT kyūjitai equivalents):
- 覚 (NOT 覺)
- 転 (NOT 轉)
- 観 (NOT 觀)
- 体 (NOT 體 in modern ja)

**Tech-context-only kanji swaps**:
- 図 → 画像 **ONLY in tech / UI context** (screenshot figures, charts in a UI). Everyday words like 図書館 / 設計図 / 窓辺 / 窓を開ける **keep standard 図 / 窓 kanji** — they are NOT on a blanket avoid list. (Lesson learned 2026-05-17 when an agent over-applied the rule and turned 図書館 into 書庫.)
- 窓 → ウィンドウ **ONLY for software UI windows**.

**Pronoun sense vs Sino-compound**:
- 自分 for "self / oneself" as pronoun
- 自己紹介 / 自己描写 / 自己肯定 — Sino-Japanese compound, keep 自己

**Chinese-character intrusions to avoid in JA text**:
- 拖 (use 運び / ドラッグ)
- 拋 (use 放 / 投げる)
- 一鍵 (use ワンタップ in UI context)

**JA chapter counter**: 第N話 (NOT 第N頁 — Nova locked this 2026-05-17). 話 is the standard counter for chapters/episodes in JA literary register.

**Editorial / magazine register**: ja chapter dek and Verbatim blocks aim at *Shippori Mincho* typography. Avoid `である` overuse; prefer measured `ました／です` or literary `ぬ／なる` only when conscious.

## 7 · EN language strategy

**Clarity over flair** (Nova explicit, 2026-05-17): she is not a native EN reader and wants the form readable, not literary-show-off. Choose plain word over archaic synonym when both work.
- Example: "weight" over "heft" for 碑文份量.

**EN chapter label drop-middle**: When EN name = EN-CAPS suffix in tri-script, drop the middle to avoid `0 · Preface · PREFACE` redundancy. Use `0 · PREFACE`. Same logic for `II · SYSTEM`. (Milestone stays 3-component because Milestones ≠ CHRONICLE.)

**HK / Cantonese idioms**: literal translations are usually fine in EN (they preserve flavor without local cultural baggage).
- 馬照跑舞照跳 → "carry on with their horses and their dance" ✓ Nova-approved

**Proper noun "Vassal"**: 宰相 → Vassal (capital V, proper noun). Already established in `en/us.html "A Letter to the Next Vassal"`.

## 9 · URL / SEO conventions

- **Clean URLs everywhere outside of internal hrefs**: canonical / sitemap / og:url / hreflang strip `.html`. Internal `<a href>` can keep `.html` (Cloudflare serves both).
- **JSON-LD `inLanguage`**: single locale string per file (not array). Each lang variant gets its own headline / description matching that locale.
- **JSON-LD `sameAs`**: LinkedIn + Instagram + GitHub `github.com/tb1982`. GitHub was stripped during the 2026-05-11 suspension and re-added 2026-06-14 after the account was restored (05-20). Order: LinkedIn → Instagram → GitHub.

## 10 · Visibility / discoverability policy

- **us.html** (Sapere Aude 2.0): sitemap-only, NOT in main nav. Humans dig via harness章末 trail-dot or ch7.prose3 inline-dot.
- **/harness/{context,constraints,entropy}**: sitemap-only, NOT in main nav. Reached via harness chapter trail-dot or i18n inline-dot.
- **Principle** (Nova 2026-05-17): 「人類無法看懂不代表爬蟲或 AI 無法看懂——我想確保人機都有閱讀權利」. Hide-from-human ≠ hide-from-machine. Default to giving both audiences access; conceal from one only with conscious reason.
