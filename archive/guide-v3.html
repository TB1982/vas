入門 · Getting Started

第一次打開 VAS

從 App Store 下載安裝後打開 VAS，你會看到桌面上有一排嶄新的浮動工具列。他可以停駐在你的桌面上，任你拖動。

如果你覺得他在桌面上有點佔位想讓他小一點——呼吸燈模式預設為關閉，請到工具列「快捷鍵說明」內手動開啟。

VAS 同時進駐 macOS 選單列上，以一個瓶子的形狀出現在你螢幕右上角。

想讓VAS完全消失？

到右上選單列點一下瓶子（左鍵），可以召喚或完全隱藏桌面上的工具列——如果你是桌面潔癖患者，我們都幫你想好了。

VAS 依然會在選單列待命。

授權與隱私

VAS 第一次截圖時會請求 macOS 的螢幕錄製權限——這是 macOS 對所有截圖類工具的規定，不可省略。如果當時沒授權，到「系統設定 → 隱私權與安全性 → 螢幕錄製」勾選 VAS，重新開啟即可。

Tauri 專屬如果你是從 Electron 版升級過來的，macOS 會把 Tauri 版視為全新的應用程式，舊的授權不會沿用。建議先到「系統設定 → 隱私權與安全性 → 螢幕錄製」移除舊的 VAS 授權，再用 Tauri 版執行一次截圖，macOS 才會跳出重新授權的提示。

所有截到的圖片、OCR 辨識出的文字、QR Code 的內容，完全在你的電腦本機處理——不上傳、不分析、不訓練。VAS 沒有伺服器來接收你的資料。

第一次截圖

按 Dock 上的 VAS 圖示，或點選單列的瓶子——不管 VAS 原本在哪，它會被「召喚」到你眼前，展開成完整的工具列。

選一個截圖模式（全螢幕 ⌥⌘1、視窗 ⌥⌘2、矩形 ⌥⌘X），框出你要的畫面——VAS 會立刻把畫面帶進編輯器，讓你繼續編織你無法單用文字描述的意念。

編輯完，用「拖曳匯出」拖出去給任何視窗或文件。

三秒內結束。

核心 · Core

系統選單中的瓶子

選單列右上角那個瓶子，是 VAS 的「系統選單」入口。兩個按法：

  • 左鍵——快速切換桌面工具列的顯示／隱藏。給桌面潔癖患者一個完全不顯示於桌面的選項。
  • 右鍵——展開系統選單，可以完整執行所有工具列功能，不用把工具列叫出來。

如果你完全不想在桌面上看到任何 VAS 的痕跡，左鍵把工具列關掉，右鍵仍可以操作工具列所有功能。

一盞呼吸燈 x 一鍵召喚

當你不用 VAS 又希望他能小一點在桌面上待機時，請開啟呼吸燈模式。

工具列會縮成一條 120×6px 的光。低調、不擋事，靜靜的等你。

而當你有事要找它時：

  • 全版本拖曳圖片靠近——工具列像捕蠅草一樣張開，接住你拖過來的圖。放開的瞬間，圖直接進編輯器；如果你剛剛還在編輯別的東西，它會收納那張、直接展開編輯器繼續。
  • Tauri 專屬拖曳圖片匯出——編輯器像黑洞一樣坍縮，給你最大的桌面空間與視野進行存檔或傳遞操作。單鍵拖曳至視窗 / 對話框 / 資料夾隨心所欲。放開的瞬間，直接展開回編輯器繼續。
  • Tauri 專屬複製網址靠近——呼吸燈張開，跳出 toast：「檢測到 ⟨網址⟩—點擊開始截圖」。點一下就開始網頁截圖流程。
  • Tauri 專屬從瀏覽器複製圖片——跳出 toast 問你「貼上圖片」，你說好就直接進編輯器。
召喚

如果你把 VAS 放在桌面某個角落後忘了它在哪——我們知道呼吸燈真的很低調——

按 Dock 上的 VAS 圖示，不管它在哪裡，都會展開成工具列狀態等你過去。

工具列

懸浮在桌面上的截圖與工具入口

全螢幕

截取整個螢幕畫面，立即進入編輯器，支持多螢幕截圖。

視窗

點選後再點擊任意視窗，只截取該視窗範圍。

矩形

拖曳選取任意矩形區域後截圖。直接框住 QR Code 會自動掃描並顯示內容。

延遲

設定 1–3 秒延遲，補抓 tooltip、hover、dropdown 等稍縱即逝的狀態。

開啟

從本機開啟現有圖片。

白板

開啟空白畫布，從零開始建立圖像內容，支持透明圖層。

批次轉

拖入多張圖片，一次轉換為 JPG、PNG、WebP 等格式，支援浮水印與尺寸調整。

單次最大 100 張圖，單檔 20mb

說明

顯示快捷鍵一覽，並提供操作手冊連結與呼吸燈開關。

編輯器

截完圖後進的第二個主要空間。編輯器分為四個操作區域：

主要功能選單左側垂直工具列。點選工具後，畫布上的操作會對應切換。目前使用中的工具以藍色高亮標示。
子屬性調整選單頂部工具列。依所選工具動態更換內容，可調整顏色、粗細、不透明度、線條樣式等屬性。
動態匯出按鈕右下角浮動按鈕。可拖曳移動位置，對應左右撇子不同使用習慣。完成後拖曳至目標應用程式即可匯出。
狀態列與磁吸工具開關左下角狀態列。顯示目前縮放比例與畫布尺寸，並提供磁吸對齊開關，可快速啟用或關閉智慧導線功能。

匯出

編輯完了，VAS 給你四條出口：

  • Drag out——直接從編輯器拖到桌面、Finder、Slack、任何一個接受圖片的 App。不儲存、不彈視窗、拖過去就傳輸完畢。
  • 複製到剪貼簿——按「複製」按鈕，圖片進剪貼簿，到別的地方按 ⌘V 貼上。
  • Tauri 專屬ShareSheet——macOS 原生分享面板，轉給 AirDrop、訊息、郵件或你的社群媒體。
  • 存檔——可儲存為 JPG／PNG／WebP／GIF／BMP／TIFF／PDF。

進階 · Advanced

煉金配方

VAS 把工具拆得很小。每一個只做一件事——但屬性可以疊。

  • 文字 × 底色 → 螢光筆
  • 文字 × 透明 + 外框 → 透明字
  • 框線 × 虛線 + 粗細 → 審稿紅圈
  • 線條 × 直角 + 虛線 → 流程連接
  • 色塊 × 漸層 + 低不透明 → 漸隱遮罩
  • 氣泡對話框 × 馬賽克 → 漫畫風迷因

你組合出來的配方，是屬於你的表達語彙。

→ 也是屬於你跟 VAS 的 Inside Language

Tauri 專屬自訂樣式 · 把配方存起來

配方不必只活在這一次截圖裡。調出完美的那一刻——把它收進你的櫃子，下次一個右鍵就喚回。

  • 新增——在調好的標註上按右鍵，選「新增為自訂樣式」，一按就存好，自動命名為「樣式X」。不打斷你的當下，要叫什麼可以晚點再說。
  • 套用——畫布上任何現有標註按右鍵，直接套上一個已存檔的樣式；或反過來，用當前這個標註取代某個舊樣式，讓配方跟著你進化。
  • 編輯——屬性欄點開自訂樣式的「編輯樣式」，要改名、要刪除都在這裡，整理你的櫃子。

✦ 許願新元素

OCR 與 QR Code

截圖不一定為了編輯。

OCR · 文字辨識

辨識出的文字預設直接放進剪貼簿，截完圖 ⌘V 就能貼到任何地方。

Tauri 專屬針對當前支援的語言市場，OCR 會做在地隱私遮蔽特化——

譬如會認得當地的身分證號、信用卡號、電話格式，辨識時做對應處理。

QR Code · 掃描

VAS 根據 QR Code 在畫面中的佔比決定行為：

  • 框得完整——判定為你主動想掃，直接跳轉到對應連結。
  • 框得鬆（部分包含）——先問你一聲「要跳轉嗎？」。
  • 完全是背景元素——不做任何動作，直接截圖進編輯器。

自訂設定

VAS 刻意做得不多設定項，因為我們相信好的工具預設就是對的。能調的幾項是：

  • 桌面工具列顯示／隱藏——選單列左鍵瓶子即可切換。
  • 呼吸燈開關——出廠預設關閉，工具列的「快捷鍵說明」裡可手動開啟。
  • Tauri 專屬自訂快捷鍵——付費版可把全螢幕截圖、視窗截圖、矩形截圖、貼入圖片等快捷鍵改成你慣用的組合。
