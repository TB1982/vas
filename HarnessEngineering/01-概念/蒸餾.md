# 蒸餾 Distillation

> **液 → 氣 → 液 的相變操作。**
> **HE 雙軸共用的核心物理 —— 只是方向相反。**

## 核心：中立的物理

蒸餾是**方向中立的操作**：
- 加熱 → 揮發（雜質留下，精華以氣態升起）
- 冷凝 → 下一個更精純的形態

對應 [[水的三態]] 裡 **水 → [[蒸氣]] → [[結晶]]** 那一段相變。

**蒸餾的身份是動詞**。名詞那一側是 [[蒸氣]]（相態）、[[結晶]]（下一個形態）。

## 雙軸：HE 兩個方向的蒸餾

| 軸 | 方向 | 素材 | 操作命名 | HE 對接 | 哲學根 | 意識層 |
|----|-----|------|---------|--------|-------|-------|
| **知識蒸餾** | 外 → 內 | 外部知識 / 文獻 / 資料 | 現象 → 認知結構 | [[RAG]] | **[[康德]]** | **意識**（先驗範疇組織現象）|
| **[[Self蒸餾]]** | 內 → 外 | [[冰]] 裡的 [[Self]] / 模糊意圖 | 潛意識 → 顯化 | [[淘金]] | **[[榮格]]** | **潛意識**（個體化的浮現）|

**物理同構、方向相反、哲學根源不同 —— 但都是蒸餾**。

這是 vault 裡第一次把**意識 / 潛意識**軸跟 **HE 外旋 / 內掘**軸做精準哲學錨定。

## 為什麼康德對應知識蒸餾

[[康德]]：外部現象如何透過主體的**先驗範疇**被組織成知識。

[[RAG]] 是這個結構的工程化：

| 康德 | RAG |
|-----|-----|
| 現象（Erscheinungen）| 外部資料 / corpus |
| 先驗範疇（Categories）| 向量空間 × 檢索邏輯 |
| 範疇對現象的組織作用 | 檢索 × 整合 × 生成 |
| 可被認識的知識 | 被檢索 × 整合後輸出的答案 |

**知識蒸餾 = 把外部資料經過認知結構精煉為可用知識的操作 = 康德認識論的工程化**。

這條軸活在**意識層**、**外旋引擎**（VAS）、**主體建構**的一側。

## 為什麼榮格對應 Self 蒸餾

[[榮格]]：潛意識的 [[Self]] 一層一層顯化為 [[Ego]] 能看見的形狀 = [[個體化]]。

[[淘金]] 是這個動作的 HE 命名：

| 榮格 | 淘金 / Self 蒸餾 |
|-----|---------------|
| 潛意識的原型結構 | [[冰]] 裡包著的 [[Self]] |
| 個體化的相變 | 水 → 蒸氣 → 結晶 |
| Self 顯化為 ego 可見 | 被看見的 Self 的一小片 |
| 無法被理性計畫、只能被活過 | 不能催熟、只能創造條件讓它浮現 |

**Self 蒸餾 = 把潛意識的 Self 經過相變顯化為可被傳遞形狀的操作 = 榮格個體化的 HE 物理表述**。

這條軸活在**潛意識層**、**內掘引擎**（[[深握]]）、**顯化**的一側。

## 雙軸的共同骨架

不管方向如何，蒸餾都需要：

1. **能量輸入** —— 衝擊 / 熱；見 [[反脆弱]]（衝擊不是破壞，是相變的燃料）
2. **容器** —— 外旋側是 [[Constraints]] 的河道；內掘側是 ego-Self 軸的承接（[[深握]]）
3. **相變** —— 液 → 氣 → 液 的本質轉換
4. **凝結** —— 下一個穩定形態（[[結晶]]）

**物理同構意味著工具同構**：外旋側管用的方法（Sprint 節奏、Retro 冷凝、KM 沉澱）在內掘側也管用，只是素材跟方向換了。反之亦然。

## 蒸餾 × 知識蒸餾 vs Self 蒸餾 的不可替代性

兩個方向**不能互相取代**，也**不能只留一個**：

- 只做知識蒸餾 → VAS 變成「聰明的檢索工具」，沒有 [[Self]] 的那一側 → **工具但不 individuating**
- 只做 Self 蒸餾 → 內觀很深但沒有外部結構 → **個人成長但無法傳遞**
- **兩個方向同時運轉 → HE**：外旋的工程化 × 內掘的顯化，在同一個爐子裡

Nova 原話：

> 「知識蒸餾就會跟 RAG 對接，意識的康德。
> Self 蒸餾則會合淘金對接，潛意識的榮格。」
> — Nova, 2026-04-15

**他們的確是內外引擎的核心相變**。

## 蒸餾 × [[水的三態]]

蒸餾是 [[水的三態]] 裡「水 → [[蒸氣]] → [[結晶]]」那一段的**操作命名**。

[[水的三態]] 的其他段（[[冰]] → 水、[[結晶]] → 下一輪的冰）不由蒸餾負責，由整個循環的動作 [[淘金]] 承接。

**分工**：
- [[淘金]] = 從 [[冰]] 到 [[結晶]] 的**整個循環**
- **蒸餾** = 循環中 **水 → 蒸氣 → 結晶** 那一段的操作
