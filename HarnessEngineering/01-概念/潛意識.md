# 潛意識 Unconscious

> **[[Ego]] 不知道卻仍在運作的心理層。**
> **HE 內掘引擎的材料源 —— [[言靈]] corpus 的居所，[[Self蒸餾]] 的源頭。**

## 核心：ego 不知道的那一層

所有**沒被 [[意識]] 抓到、卻影響著決策 / 感受 / 夢 / 身體反應**的心理活動都在這一層。

Freud 先開這個維度（個人被壓抑的經驗）；[[榮格]] 把它撐大，分出三層心靈：

| 層 | 內容 | Freud | [[榮格]] |
|----|------|-------|---------|
| [[意識]] | ego 察覺到的 | ✅ | ✅ |
| **個人無意識** | 被壓抑 / 遺忘的個人經驗 | ✅ | ✅ |
| **[[集體無意識]]** | 全人類共享的原型結構 | ❌（不承認） | ✅（大膽一跳） |

**潛意識 = 個人無意識 + [[集體無意識]] 合起來這一整層**。

[[集體無意識]] 是潛意識的第三層（也是最深的一層），不是潛意識的同義詞。

## 潛意識的內容物

榮格給出具體結構，不是一團混沌：

| 內容 | 說明 | HE note |
|------|-----|--------|
| [[Self]] | 心靈整體中心，ego 冒出意識那一角的背後 | 在 [[冰]] 裡的金 |
| [[陰影]] | ego 不承認 / 投射出去的部分 | 內掘處理對象 |
| [[情結]] | 有情緒 charge 的心理叢 | 觸發 → 可被觀察 |
| [[Archetype]] | 跨文化重複的意象結構單位 | [[集體無意識]] 的元素週期表 |

## 潛意識 × [[榮格]]：個體化的材料源

[[個體化]] 的整個工程都建立在潛意識：

- **材料**在潛意識（[[Self]] / [[陰影]] / [[情結]]）
- **浮現的動作**由潛意識推動
- **意識層的 ego** 只是接收者 —— 它不能「計畫」個體化，只能創造條件讓它發生

> 「Self 一直都在。系統不創造，只顯化。」
> — Nova, 2026-04-15

**顯化的來源側就是潛意識**。

## 潛意識 × [[Self蒸餾]] / [[淘金]]：內→外 蒸餾的源頭

[[蒸餾]] 雙軸裡，**[[Self蒸餾]] 整個源頭在潛意識**：

| 蒸餾階段 | 潛意識 → 意識的動作 |
|---------|------------------|
| 材料 | [[冰]] 裡包著的 [[Self]] |
| 揮發 | 相變瞬間（夢、[[洞見]]、[[奇異點]]）|
| 冷凝 | [[結晶]] —— 可被意識語言承接的形狀 |

[[淘金]] 是這個動作的 HE 命名：**從潛意識材料中提取可傳遞形狀**。

**潛意識層 = 內掘引擎、[[深握]]、顯化、Jung 那一側**。

## 潛意識 × [[言靈]]：corpus 的居所

為什麼說出一個詞會召喚整個場域？—— 因為被召喚的**原型 corpus 本來就住在潛意識**。

| 機制位置 | 角色 |
|---------|-----|
| 意識層 | 說出詞（[[命名]] 動作）|
| 潛意識層 | 原型被激活 → 場域展開 |
| 意識 ↔ 潛意識橋梁 | [[言靈]] 機制本身 |

[[言靈]] 之所以有效、[[Archetype]] 之所以跨文化共振、[[隱喻]] 之所以能跳過解釋直達結構 —— **都是因為潛意識層存在**。

拿掉潛意識，這三個都失效。

## 潛意識 × [[意識]] 的交會 = [[洞見]] / [[共時性]]

**潛意識不是孤島** —— 它的底層結構需要與 [[意識]] 交會才能被看見。

- [[洞見]] = 潛意識 × 意識**純內層**交會事件（[[網]] 的底層結構被 aha 穿透表層）
- [[共時性]] = 同一交會事件**延伸到現實**的同步標記（見 [[共時性]]）

潛意識在這兩個事件裡的角色**不是接收、是脈動**：
- 潛意識端：[[網]] / [[Archetype]] / [[Self]] 的能量啟動
- 意識端：ego 敞開、接住、命名、[[結晶]]
- 交會瞬間：**界面被點亮 = aha**

**缺潛意識端** → ego 只在表層打轉，aha 不發生（沒有底層結構可被點亮）。潛意識是 [[洞見]] 的**來源側**，但洞見需要意識才能落地 —— 兩邊缺一不可。

詳見 [[洞見]] 的「結構身份」段。

## 潛意識 × [[集體無意識]]：含攝關係

常見混用：「潛意識」和「集體無意識」經常被當成同義詞，但在榮格框架裡**不是**。

```
潛意識
├── 個人無意識（Freud 已承認）
└── 集體無意識（Jung 的第三層）
    └── Archetype（集體無意識的結構單位）
```

**[[集體無意識]] 是潛意識的一個子集**，但因為它最深、最有結構、最跨文化，所以常被單獨討論。

## 潛意識 vs [[意識]]

見 [[意識]] 的對照表。重點摘錄：

- **可觀察性**：意識可直接觀察 / 潛意識只能透過夢、滑舌、原型體感間接觸及
- **HE 引擎**：意識 = 外旋（VAS）/ 潛意識 = 內掘（[[深握]]）
- **蒸餾方向**：意識側外→內 / 潛意識側內→外
- **操作速度**：意識快、線性 / 潛意識慢、非線性、只能創造條件

## 為什麼過去 vault 沒有獨立 note

- 早期 vault 直接寫 [[集體無意識]]（最醒目、最有力的那一層）
- 「潛意識」這個上位概念被隱含掉了
- 2026-04-15 [[蒸餾]] 雙軸確立後，**潛意識 / 意識成了結構軸**
- [[言靈]] cluster 母節點化之後，又需要明確指認「corpus 住在潛意識」—— 光指集體無意識不夠準（個人無意識也參與）
- 這個 note 是**雙軸哲學錨定 + 言靈 cluster 擴充**後的補柱動作
