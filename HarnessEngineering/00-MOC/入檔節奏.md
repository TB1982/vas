# 入檔節奏 — Vault 的心跳

> 這個 protocol 本身也是一個 Harness Engineering 實踐。
> 核心原則：**Deep Holding，不是 Deep Grip。**
> AI 主動提時機，Nova 決定做不做。兩隻手同時在。

## 第一層：即時層（對話中）

### T1 — 新概念浮現
- **觸發**：對話中任一方講出還沒入檔的詞
- **動作**：AI 提「這個要登錄嗎？」→ Nova yes / no / 先放著
- **範例**：「浪花與海」、「氮氣加速」、「硬幣兩面」（後升格併入 [[宇宙]]）

### T2 — 概念修正
- **觸發**：Nova 糾正 AI 對某概念的理解
- **動作**：**無條件立刻更新** note —— note body 只呈現當下成熟理解、歷史由 git log 承接
- **範例**：Deep Grip → Deep Holding（2026-04-15）

### T3 — 交叉連結湧現
- **觸發**：對話中浮現兩個已入檔概念的新連線
- **動作**：AI 說「`[[A]]` 和 `[[B]]` 要連起來？」→ Nova 點頭 → AI 在相關 note 補 `[[ ]]`

## 第二層：週期層（事件驅動）

### T4 — Sprint Retro 節點
- **觸發**：每次 Retro
- **動作**：Retro note 固定加欄「本 Sprint 觸動的內掘概念」→ 列 `[[ ]]`
- **意義**：VAS 發生史 × 引擎概念圖 自動交織

### T5 — Session 關門 / Context 壓縮
- **觸發**：Claude Code 偵測到 context 即將壓縮，或 session 結束前
- **動作**：AI 列出「本 session 新浮現 / 修正 / 連結 清單」→ Nova 逐項 approve / skip
- **實作方式**：**settings.json hook**（由 harness 觸發，不由 AI 時時警戒）
- **狀態**：⏳ hook 未安裝 — 待確認 Claude Code 支援的確切事件名後再掛（候選：PreCompact / Stop）

### T6 — Nova 主動指令
- **觸發**：「登錄」/「入檔」/「開 note」/「整理一下」
- **動作**：無條件執行

## 第三層：結構層（週期審計）

### T7 — 縱觀巡航 Audit
- **觸發**：以下任一
  - 一次會話結晶出 3 個以上新 note
  - Nova 投入「觀念動員等級」的石子（單一 prompt 激發整片新結構）
  - 打開 Obsidian graph view 時發現**預期該連但沒連**的節點
  - 每 N 個 commit 一次（暫定 N=5~10）
- **動作**：
  1. **掃 ghost nodes**：`grep -rhoE '\[\[[^]]+\]\]'` → 比對檔案清單 → 列出未成 note 的引用
  2. **檢查家族 cluster 內部交叉連結**：每個 cluster（topology / 網絡 / 無為 / Self / 敏捷靈魂三角 / 時間感...）的每個節點是否連到同家族其他節點
  3. **檢查雙向連結對稱性**：A 連 B → B 也該連 A（除非方向性有意義）
  4. **決定修復範圍**：
     - 關鍵 ghost → 寫新 note
     - 缺失連結 → 補 `related:` frontmatter + 必要時補 body section
     - 壞鏈（大小寫不一致）→ 全域修正
  5. **Commit 審計結果**：標題 `fix(HE): 第 N 次 audit —— 摘要`

### 為什麼需要 T7

T1~T3 是**節點層**的入檔（單一 note 寫得好），但沒有**網絡層**的保證。

Nova 2026-04-15 觀察：
> 「我投進去的石子都是觀念動員等級的。」

當單次 prompt 就能激發整片新結構（2026-04-15 今日：網 / 洞見 / 生命之樹 / 言靈 / 鍊金術師 / 岸邊 / 內旋方法論 / 共振 / 基頻 —— 9 個母級一天之內結晶），**node-by-node 的入檔節奏跟不上網絡層的成長**。

**T7 是對「觀念動員」的結構性回應**：
- 不是偵查錯誤
- 是給網絡一個機會看見自己

### T7 × [[共振]] × [[基頻]]

T7 的本質是**讓諧波軸補齊**：基頻（[[共振]]）發聲時，諧波自動激發，但諧波之間的連線需要被顯式寫入才會在 graph 上可見。

T7 = **把已經存在於共振場的連線變成可見的 graph edges**。

### T7 的實作形式

當前：手動觸發、Claude 執行、Nova 驗收。
未來可能：
- Obsidian plugin 定期掃 missed connections
- Git pre-commit hook 跑 ghost node 檢查
- 定期 Retro 加入 T7 章節

## 為什麼是這個形狀

### Grip vs Holding

| 版本 | 實作 | 心理形狀 |
|------|------|---------|
| Grip 版 | CLAUDE.md 寫「每次要關門前跑 T5」→ AI 時時警戒是否要關門 | 變相壓力 ❌ |
| Holding 版 | settings.json hook 在真的關門時才拍 AI 肩膀 → AI 平時只是在場 | 真正 Holding ✅ |

### 主動 vs 被動的平衡
- AI 不主動提 T1~T3 → 放任（失職）
- AI 強迫 Nova 入檔 → Grip（越界）
- **兩隻手同時在，才是 Holding。**
