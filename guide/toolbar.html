手冊 · 01 — 工具列 · Toolbar

Mac 截圖，從這條工具列開始

桌面上那條浮動工具列，是 VAS 一切的起點——四種截圖模式、取色器、尺標、延遲與批次都在這裡。不用時，它縮成一盞呼吸燈。

點工具列上任一顆按鈕 · 下面就換成它的說明

全螢幕

截取整個螢幕畫面，立即進入編輯器，支援多螢幕截圖。

視窗

點選後再點擊任意視窗，只截取該視窗範圍。

矩形

拖曳選取任意矩形區域後截圖。直接框住 QR Code 會自動掃描並顯示內容。

長截圖 Tauri 專屬

自動偵測桌面前景的 Google Chrome 視窗，捲動抓取整頁網頁並自動拼接，還能自動清除重複的回頁首／置頂鈕。

＊首次使用需在「系統設定 → 隱私權與安全性 → 輔助使用」為 VAS 開啟權限

取色器 Tauri 專屬

滴管取螢幕上任一像素的精確色碼，常用顏色自動存入最近色並與編輯器色盤連動。

尺標 Tauri 專屬

在畫面上拖框自動吸附邊緣量出像素尺寸，或點兩點量點對點距離；量測線可選取刪除、可復原／重做，輸出時烘進圖裡。

開啟

從本機開啟現有圖片。

全版本PNG / JPG / WebP / TIFF / GIF / SVG，Tauri 專屬BMP

白板

開啟空白畫布，從零開始建立圖像內容，支援透明圖層。

更多

收納延遲截圖與批次轉檔。延遲可設 0–8 秒，補抓 tooltip、hover、dropdown 等稍縱即逝的狀態。

批次：單次最大 100 張圖，單檔 20mb

批次格式轉檔 · Batch Convert

一次把多張圖片轉成統一格式，並可在轉換時統一縮放、加浮水印、指定儲存位置。適合大量截圖／素材的整批處理。

加入檔案

  • 拖曳圖片到視窗，或按「選取檔案」挑選，亦可「＋新增檔案」續加。
  • 單次最多 100 張。

支援格式

  • 可讀入——PNG、JPG／JPEG、WebP、GIF、BMP、TIFF、SVG（向量圖會在程式內光柵化）。
  • 可輸出——PNG、JPG、WebP、GIF、BMP、TIFF 六種。Tauri 專屬其中 BMP 及 TIFF 僅 Tauri 支援。
  • 輸出 JPG／WebP 時可設品質（1–100%）；批次中含 SVG 時可設 SVG 輸出寬度（決定光柵化解析度）。

統一調整尺寸（選用）

  • 固定寬度或高度為指定像素，另一邊等比縮放。

浮水印（選用 · 文字與圖片可同時使用）

  • 文字浮水印——輸入文字（支援中日文）、字級、顏色、不透明度，以及九宮格位置（四角／四邊／置中）。
  • 圖片浮水印（Logo）——選一張圖、設寬度佔比（相對原圖）、不透明度、九宮格位置，並有即時預覽。
  • 兩者共用邊距設定；同時開啟時，文字在下、圖片在上。

儲存位置

  • 同原始檔案目錄，或指定目錄（自選資料夾）。
  • 可勾選「轉換完成後刪除原始檔」——原檔會移入垃圾桶（仍可復原，非永久刪除）。
  • 轉換過程會逐張顯示進度。

呼吸燈與召喚

當你不用 VAS 又希望他能小一點在桌面上待機時，請開啟呼吸燈模式。

工具列會縮成一條 120×6px 的光。低調、不擋事，靜靜的等你。

滑鼠靠近 · 點一下召喚

閒置時是一條 120×6px 的呼吸光；滑鼠靠近聚成會呼吸的光球，點一下就彈開成完整工具列。滑鼠離開 5 秒後，它又悄悄收合回呼吸光——桌面上的完整體驗都在這裡。

而當你有事要找它時：

  • 全版本拖曳圖片靠近——工具列像捕蠅草一樣張開，接住你拖過來的圖。放開的瞬間，圖直接進編輯器；如果你剛剛還在編輯別的東西，它會收納那張、直接展開編輯器繼續。
  • Tauri 專屬拖曳圖片匯出——編輯器像黑洞一樣坍縮，給你最大的桌面空間與視野進行存檔或傳遞操作。單鍵拖曳至視窗 / 對話框 / 資料夾隨心所欲。放開的瞬間，直接展開回編輯器繼續。
  • Tauri 專屬從瀏覽器複製圖片——跳出 toast 問你「貼上圖片」，你說好就直接進編輯器。或直接用自定義快捷鍵貼入編輯器也可以。
召喚

如果你把 VAS 放在桌面某個角落後忘了它在哪——
我們知道呼吸燈真的很低調——

按 Dock 上的 VAS 圖示，不管它在哪裡，
都會展開成工具列狀態等你過去。

系統選單中的瓶子

選單列右上角那個瓶子，是 VAS 的「系統選單」入口。兩個按法：

  • 左鍵——快速切換桌面工具列的顯示／隱藏。給桌面潔癖患者一個完全不顯示於桌面的選項。
  • 右鍵——展開系統選單，可以完整執行所有工具列功能，不用把工具列叫出來。

如果你完全不想在桌面上看到任何 VAS 的痕跡，左鍵把工具列關掉，右鍵仍可以操作工具列所有功能。
